# Moss Block

## Hard Data
* **Spawns/Found/Crafted:** Naturally generates in [[lush_caves]] and [[trial_chambers]].
* **Requirements:** [[hoe]] for fastest mining.
* **Stats/Drops:** Drops itself; can be spread using [[bone_meal]].

## The Strategy
Moss blocks can be used to quickly terraform areas. Applying [[bone_meal]] to a moss block spreads it to surrounding stone or dirt, creating a lush carpet.

> [!tip] Efficiency Tip
> Place [[sniffer]] eggs on top of a moss block to make them hatch ==twice as fast==.
