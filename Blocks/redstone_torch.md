# Redstone Torch

## Hard Data
* **Spawns/Found/Crafted:** Crafted using ==1== [[redstone_dust]] and ==1== [[stick]].
* **Requirements:** None.
* **Stats/Drops:** Provides a constant redstone signal to adjacent blocks.

## The Strategy
The [[redstone_torch]] is a fundamental power source in redstone circuitry. It can be used as a torch for light or as an inverter (NOT gate) when placed behind a block.

> [!tip] Efficiency Tip
> Use [[redstone_torch]]s to power [[powered_rail]]s without needing a complex wiring system.
